PGEOnlineIntegrationServiceSoapBinding

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
PGEOnlineIntegrationServiceSoapBinding service = new PGEOnlineIntegrationServiceSoapBinding(null,"http://localhost:8080/software-engeneering-wildfly-archetype-ejb/OnlineIntegration",30000);
service.login();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
PGEOnlineIntegrationServiceSoapBinding service = new PGEOnlineIntegrationServiceSoapBinding(null,"http://localhost:8080/software-engeneering-wildfly-archetype-ejb/OnlineIntegration");
service.login();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
PGEOnlineIntegrationServiceSoapBinding service = new PGEOnlineIntegrationServiceSoapBinding();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.login();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
PGEOnlineIntegrationServiceSoapBinding service = new PGEOnlineIntegrationServiceSoapBinding();
service.enableLogging=true;
service.login();
